# Build a Complete PDF Form Filler Web Application

Create a production-ready PDF Form Filler web application using **Python 3** and **Flask**.

## Objective

The application should allow users to:

1. Upload an interactive PDF (AcroForm).
2. Automatically detect all fillable fields in the PDF.
3. Generate an HTML form dynamically based on those fields.
4. Allow users to enter values into the generated form.
5. Fill the original PDF with the submitted values.
6. Preview the completed PDF in the browser.
7. Download the completed PDF.
8. Optionally flatten the PDF so the filled values become permanent.

---

## Features

### Dashboard

A clean Bootstrap interface with:

* Upload PDF button
* Recent uploaded files
* Generate Form button
* Download button
* Preview button

---

### PDF Upload

Users can upload only PDF files.

Requirements:

* Validate file extension.
* Validate MIME type.
* Limit upload size to 50 MB.
* Store uploads in the uploads directory.
* Display success/error messages.

---

### Read PDF Fields

Use pypdf to read AcroForm fields.

Automatically extract:

* field name
* field type
* page number

Support:

* Text fields
* Checkboxes
* Radio buttons
* Dropdowns
* List boxes
* Multiline text

---

### Dynamic HTML Form

Generate Bootstrap forms automatically.

Examples:

Text field →

```
<input class="form-control">
```

Checkbox →

```
<input type="checkbox">
```

Dropdown →

```
<select class="form-select">
```

Radio →

Bootstrap radio buttons.

The application should automatically choose the correct HTML control.

---

### Fill PDF

When the user submits the form:

* Read all submitted values.
* Fill every page of the PDF.
* Preserve existing formatting.
* Save the completed PDF in the output folder.

---

### Preview

Embed the filled PDF using:

```
<iframe>
```

or

```
<object>
```

Include Download and Back buttons.

---

### Download

Allow downloading the generated PDF with the filename:

